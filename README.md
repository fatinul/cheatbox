## Installation

| Method | Command |
| :----- | :------ |
| pipx *(recommended)* | `pipx install mvw` |
| uv | `uv tool install mvw` |
| pip | `pip install mvw` |

### From Source (recommended: uv)

```bash
git clone https://github.com/fatinul/cheatbox
cd cheatbox
uv run -m cheatbox.main
```

## Usage

```bash
cheatbox
```
